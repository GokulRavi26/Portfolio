Contact Manager

An web application developed using ReactJS and MySQL to efficiently organize and manage contact information. It features role-based access where the teachers can add, delete, and view student contacts, while students can only view and edit their own profiles. The system supports full CRUD operations and includes advanced search functionality, allowing users to quickly retrieve contacts based on specific criteria such as name, department, and domain.
