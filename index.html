Contact Manager

An web application developed using ReactJS and MySQL to efficiently organize and manage contact information. It features role-based access, distinguishing between student and teacher profiles. Teachers can add, delete, and view student contacts, while students can only view and edit their own profiles. The system supports full CRUD operations and includes advanced search functionality, allowing users to quickly retrieve contacts based on specific criteria such as name, department, and domain.

Library Management

A web application built with Python and MySQL to streamline book inventory management, user accounts, and transaction tracking. It offers features for borrowing, returning, and searching books, ensuring a user-friendly and responsive experience. The system was developed using Jupyter Notebook for efficient data processing and integration.

AI-Powered Energy Demand Forecasting System

An advanced AI-driven software that utilizes machine learning, specifically a polylinear regression model built with the scikit-learn library, seamlessly integrating with the existing energy management system. The AI model predicts energy demand for the coming days with 85% accuracy, enabling optimized adjustments in energy production based on forecasted needs. It factors in variables like historical demand data, weather patterns, socioeconomic factors, holidays, events, and energy policies to ensure precise and reliable predictions.
